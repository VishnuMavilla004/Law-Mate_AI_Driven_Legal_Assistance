# ⚖️ Law-Mate: AI-Driven Legal Assistance

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/Python-3.10%2B-brightgreen)
![React](https://img.shields.io/badge/Frontend-Vite%20%2B%20React-61dafb)

**Law-Mate** is an intelligent legal assistant designed to simplify complex legal documentation and provide AI-driven insights. This project integrates a modern web frontend with a powerful Machine Learning backend to analyze legal queries and provide guidance.

---

## ✨ Key Features
* 🔍 **AI Document Analysis:** Automatically extract and summarize key points from legal texts.
* 🤖 **Smart Legal Chatbot:** Get instant answers to common legal questions using our ML service.
* 📁 **Case Management:** Organize and track legal documents in a structured dashboard.
* ⚡ **Real-time Processing:** Fast and responsive UI built with Vite and React.

---

## 🛠️ Tech Stack

### Frontend
* **React.js** (Vite)
* **Tailwind CSS** (for modern UI)
* **TypeScript**

### Backend & ML
* **Python**
* **Machine Learning Service** (scikit-learn / Transformers)
* **SQLite** (Database)
